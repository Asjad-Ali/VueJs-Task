# vuejs-task

## Project setup
```
npm install
```
## Start Json-server run this commond on cmd in project file
```
json-server --watch src/data/db.json
```

```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
